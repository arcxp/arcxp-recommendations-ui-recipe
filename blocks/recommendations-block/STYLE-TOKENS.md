## Style Tokens

This block provides the following style tokens for customization.

| **Token**                        | **Description**                                              |
| -------------------------------- | ------------------------------------------------------------ |
| recommendations               | The main container element for the recommendations carousel. |
| recommendations-card          | A card element within the carousel for a single item.        |
| recommendations-card-category | The category/section label on a card.                        |
| recommendations-card-title    | The headline element on a card.                              |
| recommendations-card-author   | The author byline on a card.                                 |
| recommendations-card-premium  | The premium ribbon on a card.                                |
